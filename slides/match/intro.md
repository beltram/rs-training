## patterns

* patterns are a combination of the following
    * Literals
    * Destructured arrays, enums, structs, or tuples
    * Variables
    * Wildcards
    * Placeholders
* To use a pattern, we compare it to some value. If the pattern matches the value, we use the value parts in the code
